# Analysis Modules

totoLABO Phase2 の分析モジュール置き場。

## Modules

- analyze_scores.py
  - 得点分析
  - ホーム/アウェイ平均得点
  - 総得点分布
  - 最頻スコア

- analyze_streaks.py
  - 連勝・連敗分析

- analyze_form.py
  - 直近5試合/10試合フォーム分析

- analyze_h2h.py
  - 対戦成績 Head-to-Head 分析

- analyze_upsets.py
  - 番狂わせ分析

- analyze_payouts.py
  - 高配当分析
